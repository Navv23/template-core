# Template Core

A template repository for building simple, fast Python tools and CLI apps.

---

## Setup

```bash
git clone <repo-url>
cd <repo-name>
```

---

## Structure

```
core/      # main code
data/      # any static assets
io/        # input/output classes
jobs/      # job classes
tests/     # tests
utils/     # utility functions

main.py    # entry point class
```
---

## Workflow

Change the repository to your desired name.
